## 🚀 Overview

**BrainGuard** isn't just another Pomodoro timer. It is a system-level shield designed to block digital distractions, enforce deep work sessions, and rebuild your attention span from the ground up. 

This repository contains the **Interactive Web Demo / Landing Page** built to showcase the UI/UX and core features of the upcoming BrainGuard Mobile App.

## ✨ Core Features

*   🛡️ **Forced Focus Session:** A powerful timer equipped with a *Strict Mode*. Once activated, distracting apps are locked at the OS level and the interface shifts to a "focus zone" mode.
*   📊 **Deep Habit Analytics:** Track your consistency with beautiful weekly charts. Monitor how much time you spend in deep work and how many distractions were successfully blocked.
*   ⚡ **Dynamic Protection Levels:** Customize your strictness:
    *   **Light:** Mutes notifications.
    *   **Normal:** Blocks apps with a "Forfeit" option for emergencies.
    *   **Hardcore:** Un-cancellable sessions. Requires a device reboot to bypass.
*   📱 **Thumb-Zone Ergonomics:** All critical navigation elements are situated at the bottom of the screen for seamless one-handed operation.

## 🛠️ Tech Stack

This interactive web presentation is crafted using modern front-end technologies:

*   **[React 18](https://react.dev/)** + **[Vite](https://vitejs.dev/)** (Lightning fast dev environment)
*   **[Tailwind CSS v4](https://tailwindcss.com/)** (Utility-first styling with premium glassmorphism effects)
*   **[Framer Motion](https://www.framer.com/motion/)** (Fluid layout animations and scroll reveals)
*   **[Lucide React](https://lucide.dev/)** (Clean and consistent iconography)

## 💻 Running Locally

Want to test the demo locally on your machine?

```bash
# 1. Clone the repository
git clone https://github.com/Hanya-Faiz/BrainGuard-Web.git

# 2. Navigate into the directory
cd BrainGuard-Web

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Visit `http://localhost:5173` in your browser to interact with the demo.
